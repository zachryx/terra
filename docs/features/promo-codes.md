# Promo Codes & Discounts

## Overview

Create and manage promotional codes for marketing campaigns.

## Discount Types

- Percentage off (e.g., 20% off)
- Fixed amount off (e.g., $10 off)
- Buy X get Y free
- Early bird discounts

## Code Configuration

- Custom code creation
- Auto-generated codes
- Case sensitivity options
- Usage limits (total uses, per user)
- Expiry dates
- Event/ticket type restrictions

## Management

- Code activation/deactivation
- Usage tracking
- Performance analytics
- Duplicate code prevention

## Restrictions

- Minimum purchase amount
- Specific ticket types
- Single use per user
- New user only codes
