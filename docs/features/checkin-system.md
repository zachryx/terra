# Check-in System

## Overview

Efficient event entry management with QR code scanning.

## Features

### QR Code Tickets

- Unique QR code per ticket
- Downloadable PDF tickets
- Mobile ticket display
- Offline QR codes

### Scanning

- Mobile app scanning
- Web-based scanning
- Manual code entry
- Bulk check-in support

### Check-in Features

- Real-time check-in timestamps
- Duplicate scan prevention
- Photo capture at entry (optional)
- Temperature screening (optional)

### Offline Capability

- Offline scanning mode
- Sync when online
- Queue management

### Statistics

- Check-in rate
- Arrival time distribution
- Entry lane efficiency
- Real-time attendee count
