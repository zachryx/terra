# Event Management

## Overview

Create and manage events with full details and media support.

## Features

### Event Creation

- Event name and description
- Date and time (start/end)
- Venue/location with address
- Capacity limits
- Event category (conference, concert, workshop, etc.)

### Event Status

- Draft (work in progress)
- Published (visible to users)
- Cancelled
- Completed (past events)

### Media Support

- Cover image/banner upload
- Multiple gallery images
- Video embedding (YouTube/Vimeo)

### Multi-day Events

- Support for multi-day events
- Different schedules per day
- Day-specific ticket types

### Event Settings

- Visibility (public/private)
- Guest limit per order
- Waitlist when sold out
- Refund policy
